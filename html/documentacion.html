Documentación técnica

Definición del Sistema

La aplicación de escritorio tiene como objetivo la gestión integral de granjas mediante el registro, seguimiento y control de grupos de animales, sus animales, los productos que consumen y los registros de consumo asociados. Se trata de una herramienta orientada a optimizar el control de recursos, adaptada para el seguimiento de diferentes tipos de granjas ganaderas.

Ha sido desarrollada en Java utilizando Hibernate para la persistencia de datos y opera sobre un servidor GlassFish con soporte para envío de notificaciones vía correo electrónico mediante un servidor SMTP. La arquitectura está pensada para facilitar que los usuarios, los administradores de la granja, colaboren sobre los mismos datos de gestión.

Descripción Funcional

El sistema está estructurado en torno a cuatro entidades principales:

  • Grupos de Animales: Representan conjuntos de animales organizados por criterios funcionales o espaciales (por ejemplo, especie, área de la granja). Cada grupo contiene una descripción, área asignada, fecha de creación, y puede estar gestionado por múltiples usuarios (managers).
  • Animales: Cada animal tiene atributos como nombre, subespecie, especie, fecha de nacimiento, y un consumo mensual estimado. Se encuentran vinculados de forma única a un grupo de animales. Los usuarios pueden registrar nuevos animales, modificarlos, eliminarlos y filtrarlos por criterios como especie, fecha de nacimiento grupo de animal al que pertenecen.
  • Productos: Elementos consumibles por los animales, tales como alimentos, suplementos o medicamentos. Cada producto tiene nombre, consumo mensual estimado, precio, stock disponible y proveedor asociado.
  • Consumos: Registros que indican cuánto de cada producto ha sido consumido por un grupo de animales en una fecha determinada. Esta entidad relaciona productos y grupos de animales mediante una clave compuesta y permite llevar el seguimiento.

Funcionalidades principales del sistema:

CRUD completo (crear, leer, actualizar, eliminar) para todas las entidades: animales, grupos, productos y consumos. Generación de reportes imprimibles de cada entidad (por ejemplo, listado de animales de un grupo con todos sus datos). Gestión multiusuario con posibilidad de que varios usuarios colaboren sobre los mismos grupos.

Identificación de Usuarios

El sistema está destinado a ser utilizado por los siguientes perfiles de usuario: Administradores de Granja / Managers: Usuarios registrados que gestionan los datos de uno o varios grupos de animales. Tienen acceso completo al sistema (CRUD), pueden generar reportes, supervisar consumos y productos, y administrar la información de los animales. Cada grupo puede estar vinculado a múltiples managers, permitiendo una gestión colaborativa.
